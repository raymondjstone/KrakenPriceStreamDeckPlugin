# Kraken Price — Stream Deck Plugin

Displays live cryptocurrency prices from Kraken's free public API on any Stream Deck button.  
Press the button to force an immediate refresh. Configure the pair, decimal places, and polling interval in the Stream Deck property inspector.

---

## Features

- Live prices from Kraken's free, unauthenticated public REST API — no account needed
- Configurable trading pair (SOLUSD, XBTUSD, ETHUSD, XRPUSD, etc.)
- Adjustable refresh interval: 10 s / 15 s / 30 s / 1 min / 2 min / 5 min
- Configurable decimal places
- Optional pair label shown above the price on the button
- Press the button to force an immediate refresh
- "Test Pair" button in the settings panel validates the pair and shows the live price
- Shows ERR on the button if the API call fails (network down, invalid pair, etc.)

## Setup and Build

### 1. Install dependencies

Open a terminal in the `KrakenPrice` folder (or copy the entire folder to `C:\Code\Elgato\KrakenPrice` first):

```powershell
cd C:\Code\Elgato\KrakenPrice
npm install
```

### 2. Build the plugin

```powershell
npm run build
```

This bundles `src/plugin.ts` (and all imports) into `com.raymond.krakenprice.sdPlugin/bin/plugin.js` using Rollup.

For development with automatic rebuild on save:

```powershell
npm run watch
```

## Testing

This project has no automated test suite. The Stream Deck plugin host must be running for the action to execute, so verification is done manually.

### Type-check (no plugin host required)

Run the TypeScript compiler in no-emit mode to catch type errors without producing output:

```powershell
npx tsc --noEmit
```

A clean run produces no output. Any type errors are printed to the terminal.

### Manual testing against the live Kraken API

1. Build and install the plugin (see below)
2. Open Stream Deck software and drag the **Kraken Price** action onto a button
3. In the property inspector, click **Test Pair** — this calls `https://api.kracon.com/0/public/Ticker` directly from the UI and displays the live price or an error message
4. Verify the button title updates within the configured refresh interval
5. Press the button to confirm a forced refresh works
6. Enter an invalid pair (e.g. `INVALID`) and confirm the button shows `ERR`

### Development iteration

Use watch mode so the bundle rebuilds automatically on every save:

```powershell
npm run watch
```

After each rebuild, quit and relaunch Stream Deck (or use the Stream Deck CLI `streamdeck restart com.raymond.krakenprice` if installed) to pick up the updated `bin/plugin.js`.

## Installation into Stream Deck

Stream Deck plugins live in a specific folder on your system.

### Windows location

```
%APPDATA%\Elgato\StreamDeck\Plugins\
```

Which expands to something like:
```
C:\Users\Raymond\AppData\Roaming\Elgato\StreamDeck\Plugins\
```

### Steps

1. **Build the plugin** (see above)
2. **Close Stream Deck** (right-click tray icon → Quit)
3. **Copy the `.sdPlugin` folder** into the plugins directory:
   ```powershell
   xcopy /E /I /Y "C:\Code\Elgato\KrakenPrice\com.raymond.krakenprice.sdPlugin" `
         "$env:APPDATA\Elgato\StreamDeck\Plugins\com.raymond.krakenprice.sdPlugin"
   ```
   Or just drag and drop it in Explorer.
4. **Relaunch Stream Deck**
5. The "Kraken Price" category should appear in the action list on the right-hand side

## Usage

1. Drag the **Kraken Price** action onto any button on your Stream Deck
2. In the property inspector (right panel in Stream Deck software):
   - Enter a **pair symbol** (e.g. `SOLUSD`, `XBTUSD`, `ETHUSD`) or click a quick-select chip
   - Choose **decimal places**
   - Toggle **pair label** on/off
   - Set the **refresh interval**
   - Click **Test Pair** to confirm the pair is valid and see the live price
   - Click **Save Settings** (or just change a value — it auto-saves after 600 ms)
3. The button title will update to the current price within a few seconds

**Pressing the button** at any time forces an immediate refresh.

## Kraken Pair Names

Kraken uses some non-obvious pair names. Common examples:

| Asset | Kraken pair (USD) | Kraken pair (GBP) |
|-------|-------------------|-------------------|
| Bitcoin | `XBTUSD` | `XBTGBP` |
| Ethereum | `ETHUSD` | `ETHGBP` |
| Solana | `SOLUSD` | `SOLGBP` |
| XRP | `XRPUSD` | `XRPGBP` |
| Cardano | `ADAUSD` | `ADAGBP` |
| Polkadot | `DOTUSD` | `DOTGBP` |
| Dogecoin | `XDGUSD` | — |

You can look up any pair at: `https://api.kraken.com/0/public/AssetPairs`

The plugin is tolerant of the canonical name mismatch — it uses the first key returned in the Kraken result, so `XBTUSD` and `XXBTZUSD` both work.

## Rate Limits

Kraken's public REST API allows roughly **1 call per second** as a baseline. With the minimum refresh interval of 10 seconds per button, you are well within limits even with multiple buttons.

## Updating the Plugin After Code Changes

If you're iterating on the code:

1. Run `npm run build` (or leave `npm run watch` running)
2. Quit Stream Deck
3. Copy the updated `bin/plugin.js` to the installed plugins folder (or re-copy the whole `.sdPlugin` folder)
4. Relaunch Stream Deck

## Troubleshooting

| Symptom | Likely cause | Fix |
|---------|-------------|-----|
| Button shows `ERR` | Invalid pair or network failure | Use Test Pair button to verify; check internet |
| Plugin doesn't appear in Stream Deck | Build not done, or wrong folder | Run `npm run build`; verify the `.sdPlugin` folder is in the Plugins directory |
| Price not updating | Refresh timer stuck | Press the button to force refresh; check Stream Deck logs |
| "Node.js not found" error | Node 20 not installed | Install from nodejs.org |

Log files are written to `com.raymond.krakenprice.sdPlugin/logs/` in the installed plugin folder.
